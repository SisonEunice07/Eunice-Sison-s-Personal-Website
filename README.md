## 📖 Installation & Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/SisonEunice07/Eunice-Sison-s-Personal-Website.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Eunice-Sison-s-Personal-Website
   ```

3. Open `index.html` in your browser.
